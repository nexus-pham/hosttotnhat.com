# AZDIGI Satellite Astro Template

Astro SSG template converted from the React/Vite version for SEO-first satellite websites.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Structure

- `src/data/site-config.ts` — configurable brand/site data
- `src/content/blog/` — blog markdown content collection
- `src/layouts/` — base and blog layouts
- `src/components/` — reusable UI blocks
